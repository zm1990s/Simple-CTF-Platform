# 快速开始：导出导入功能

## 使用导出功能

### 导出单个竞赛

1. 登录管理员账号
2. 进入 **管理** → **管理竞赛**
3. 找到要导出的竞赛
4. 点击该竞赛行的 **下载图标** 按钮
5. 浏览器会自动下载 JSON 文件

### 导出所有竞赛

1. 登录管理员账号
2. 进入 **管理** → **管理竞赛**
3. 点击页面顶部的 **导出全部** 按钮
4. 浏览器会下载包含所有竞赛的 ZIP 文件

### 导出单个题目

1. 登录管理员账号
2. 进入 **管理** → **管理题目**
3. 找到要导出的题目
4. 点击该题目行的 **下载图标** 按钮
5. 浏览器会自动下载 JSON 文件

## 使用导入功能

### 导入竞赛

1. 登录管理员账号
2. 进入 **管理** → **管理竞赛**
3. 点击页面顶部的 **导入** 按钮
4. 点击 **选择 JSON 文件** 并选择之前导出的竞赛 JSON 文件
5. 点击 **导入** 按钮
6. 成功后会显示导入的竞赛名称和题目数量
7. 新导入的竞赛状态为 **草稿**，需要手动启动

## 注意事项

1. **导入的竞赛为草稿状态** - 需要在管理界面手动启动
2. **题目文件不包含在内** - 如果题目有描述图片或附件，需要手动上传
3. **检查导入结果** - 导入后请检查竞赛和题目内容是否完整
4. **文件格式** - 只支持导入 JSON 格式的竞赛导出文件

## 常见问题

**Q: 导出的文件在哪里？**
A: 导出的文件会自动下载到浏览器的默认下载目录。

**Q: 可以编辑导出的 JSON 文件吗？**
A: 可以，但请确保 JSON 格式正确，并保留所有必需字段。

**Q: 导入会覆盖现有竞赛吗？**
A: 不会，导入会创建新的竞赛，不会影响现有数据。

**Q: 如何备份所有竞赛？**
A: 使用 **导出全部** 功能，会将所有竞赛打包成一个 ZIP 文件。

**Q: 题目文件怎么办？**
A: 当前版本不支持导出题目附件，需要手动备份和上传。未来版本会支持。
